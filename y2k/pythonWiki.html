Branching

  • main branch is protected to avoid pushing anything to the main branch directly.
  • Any trio of issue, branch, and merge request are interconnected, for example:

Issue #1: `Implement initial` is associated with a branch called ImplementInitial and merge request: #1 - Implement initial. The merge request #1 - Implement initial closes the issue #1 - Implement initial by requesting the resolving of the branch ImplementInitial to main.
